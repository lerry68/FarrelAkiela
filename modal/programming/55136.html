⠀⠀⠀Membuat sebuah aplikasi website klinik menggunakan Java Web MVC. Fitur-fitur nya antara lain:

  • Fitur Login dan Daftar Akun.
  • Fitur CRUD Dokter, Spesialis, User (Admin).
  • Fitur CRUD Pasien (Doctor).
  • Fitur Melihat dan Mengonsultasi Pasien (Doctor).
  • Fitur Membuat Konsultasi Masalah Penyakit (Patient).

Saya membuat website ini menggunakan bahasa pemrograman Java dengan sistem MVC, serta menggunakan IDE Apache Netbeans.
